# GearShift — Car Dealership Inventory System

GearShift is a full-stack Car Dealership Inventory System built with the MERN stack using Test-Driven Development (TDD), Clean Architecture, and Extreme Programming (XP) practices. It features JWT authentication, role-based access control, vehicle inventory management, purchase and restock workflows, interactive OpenAPI/Swagger documentation, and a production-ready developer experience.

---

## 🚀 Quick Start (Backend)

Navigate to the `backend/` directory to get started immediately:

```bash
cd backend
npm install
npm run seed
npm run dev
```

### Interactive API Documentation (Swagger)
Once the backend server is running, open your browser to interact with the API:
- **Swagger UI**: [http://localhost:5000/api-docs](http://localhost:5000/api-docs)
